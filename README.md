# Multi Client TCP Server

Simple multi-client TCP server using C socket programming.

## Features

* Multiple client handling
* TCP communication
* fork() system call
* Concurrent server
* Makefile build system

## Compile

```bash id="jlwmfd"
make
```

## Run Server

```bash id="olvb3d"
./server
```

## Run Client

```bash id="c5k4ik"
./client
```
